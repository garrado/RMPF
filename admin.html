🔑 Token do GitHub (Repositório VISA Privado)

Configure um Personal Access Token (PAT) com permissão de leitura no repositório privado garrado/VISA. Este token é necessário para importar os CSVs de inspeções, auditorias e a tabela CNAE. Crie um token Fine-Grained com permissão Contents → Read-only.

🧬 Tabela CNAE / Complexidade

Copia a tabela de complexidade CNAE do VISA para o Firestore. Execute este procedimento uma vez antes de usar a importação de inspeções, e repita sempre que a tabela cnae.csv do VISA for atualizada. A operação é segura e pode ser repetida.
